Начать сессию

Передача данных звуком без сети (FDD): одна сторона — A (нижняя полоса), другая — B (верхняя). Запустите на двух устройствах рядом и выберите разные роли.

Сообщения пойдут звуком. CSS надёжный, но медленный (~50 бит/с); OFDM быстрее. Auto сам откатывается на CSS при помехах.
Инструменты канала

Самотест модема (одно устройство)

Проигрывает кадр в динамик и тут же слушает микрофоном — проверяет, работает ли передача через реальный звук на этом устройстве. Прибавьте громкость и не заглушайте микрофон.

Проверка канала (это устройство)

~2 секунды тишины, затем колонки проиграют тестовый сигнал — не закрывайте микрофон. Оценка SNR и рекомендация режима.

Обнаружение устройств (по звуку)

Без сети — устройство слушает и проигрывает свой никнейм. Запустите на нескольких устройствах рядом (~20 c на поиск).
